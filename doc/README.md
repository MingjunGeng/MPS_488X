## MPS488X_CMD.txt  
Introduce how to use commands to change mac address and serial port number
### --- 0x01 connect MPS488X 
### --- 0x02 change MAC address and  Serial Number
### --- 0x03 Update Normal Mode FW
### --- 0x04 Update Recovery Mode FW
